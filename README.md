# Credit Card Approval Prediction

## Project Description
This project predicts whether a credit card application should be approved or rejected using Machine Learning techniques. The application is built using Python, Scikit-learn, XGBoost, and Flask.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- XGBoost
- Flask
- HTML/CSS
